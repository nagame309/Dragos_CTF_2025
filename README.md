# Dragos-CTF-2025

本儲存庫主要用於保存 Dragos CTF 2025 的題目內容與解題紀錄。
若題目包含需要額外下載的檔案，將會一併放置於 [Google Drive](https://drive.google.com/drive/folders/1lTWz-7PUz86P_oANkoxhRbVwL9DSi5ul?usp=drive_link)。

### 檔案保存與加密方式
為確保檔案安全性，所有大型或敏感附件均採用以下方式進行加密與壓縮：
* 加密方式：AES-256
* 壓縮格式：7z
* 解壓密碼：`otex`

請確認您的系統已安裝 7z，並使用上述密碼解壓對應檔案。
