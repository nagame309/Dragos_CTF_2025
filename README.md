## 檔案保存與解壓說明

為確保安全性，Google Drive 中的所有附件均已加密與壓縮，建議使用 7-Zip 進行解壓。

### 解壓資訊
* **壓縮格式**: `.7z`
* **加密算法**: AES-256
* **解壓密碼**: `otex`

### 注意事項
1. 請確認您的系統已安裝 7-Zip 或其他支援 7z AES-256 的解壓縮軟體。
2. 部分題目包含真實或模擬的惡意軟體樣本，解壓後可能會被防毒軟體刪除。建議在 VM 或 Sandbox 中進行解壓與分析，或將解壓路徑暫時加入白名單。
